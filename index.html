Integração Inteligente

Os componentes trabalham em conjunto para garantir que nenhum dado seja perdido. O WCF Service atua como ponto de entrada, o ClickHouse oferece performance analítica, o SQL Server garante persistência, e a FILA WCF proporciona resiliência através de reprocessamento automático.

Parâmetros de Configuração

Configurações críticas do sistema

Parâmetro ClickHouse.ATIVADO

ATIVADO = true
ClickHouse operacional
  • Sistema tenta enviar para ClickHouse
  • Sucesso: ClickStatus = 0
  • Erro: ClickStatus = 1 + fila
ATIVADO = false
ClickHouse inativo
  • Sistema não tenta enviar
  • Define: ClickStatus = 99
  • Mantém no SQL, não vai para fila

HTTP 202 Accepted

Retornado pelo ClickHouse após receber dados. Indica que a requisição foi aceita para processamento assíncrono.
